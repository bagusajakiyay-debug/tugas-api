# tugas-api

Website sederhana untuk menampilkan data Pokémon dari PokeAPI.

## Menjalankan

1. Buka `index.html` menggunakan browser.
2. Gunakan kotak pencarian untuk mencari Pokémon berdasarkan nama atau ID.
3. Klik tombol "Muat lebih banyak Pokémon" untuk menambahkan daftar awal.

## File utama

- `index.html` — halaman frontend.
- `styles.css` — gaya tampilan.
- `script.js` — logika fetch dan render data dari https://pokeapi.co/.
